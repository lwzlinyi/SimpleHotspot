# Car Hotspot Xposed Module

这是一个基于Xposed框架的车载蓝牙自动热点开启模块。

## 功能
- 检测到连接特定蓝牙设备后自动开启热点
- 时间窗口控制（默认8:00-20:00）
- 配置化参数

## 特性
- 监听蓝牙连接事件
- 自动热点控制
- 时间窗口管理
- 配置化设置

## 配置
- 目标蓝牙MAC地址：BC:82:5D:A4:D3:E1
- 时间窗口：08:00-20:00
- 热点SSID：CarHotspot
- 热点密码：12345678

## 使用方法
1. 安装Xposed框架
2. 安装本模块APK
3. 在Xposed Installer中激活模块
4. 重启设备
5. 连接目标蓝牙设备即可自动开启热点

## 权限要求
- 蓝牙权限
- Wi-Fi热点控制权限
- 系统设置权限

## 注意事项
- 需要root权限
- Android 8+需要额外权限
- 部分API需要系统签名
